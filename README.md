# 🧮 Simple Calculator (Python CLI Project)

## 📌 Project Description
This is a simple Command Line Calculator built using Python.  
It allows users to perform basic arithmetic operations like:

- ➕ Addition  
- ➖ Subtraction  
- ✖️ Multiplication  
- ➗ Division  

The program runs in a loop so users can perform multiple calculations without restarting.

---

## 🚀 Features
- User-friendly command-line interface  
- Supports continuous calculations using a loop  
- Handles division by zero errors  
- Accepts decimal (float) inputs  
- Simple and beginner-friendly code
